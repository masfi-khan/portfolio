MEET MASFI KHAN

Est. 2000 -- Queens, New York -- Computer Science -- Writing


Masfi Khan currently attends Queens High School for the Sciences at York College. Despite her school being STEM-centric, Masfi is passionate about writing poetry in order to uncover the layers of herself and the ever-changing world. She enjoys being founder and editor-in-chief of For the Sonorous, a literary journal for women and non-binary people of color. This summer, Masfi is thrilled to participate in the Girls Who Code Summer Immersion Program, where she exposes herself to computer science and joins a close-knit sisterhood. She believes that technology is integral to our society's modernization and that women need to have a growing input in its direction. She likes singing along off-key to songs, intersectional feminism, and really great books.

MY Girls Who Code Reflections

July 19, 2017
- After being introduced to both Scratch and Python, I noticed some similarities and differences between them. Like Scratch, Python also allows people to include variables, loops, and colors. The largest difference is whit the syntax--Scratch has pre-made building blocks whereas Python has guidelines for its code. Though Python is harder to learn, it gives me more control over my projects.
- I was quite surprised and confused when I first started using Python since I had to learn things like indenting the code and using different types of punctuation. To help remember the new aspects, I related Python back to Scratch. For example, in Scratch, loops are made by repeating things and arrows indicate the instructions, and Python indents those instructions.
July 21, 2017
- I learned about data structures in python today and yesterday. Since a list contains a set of data, a for-loop can be used to select the items from that list and perform a function for them. I found out that there can be lists within lists and that there is a format for extracting a specific item from elements using indices. I find it useful because it keeps the data organized and available for coding.
- I worked best with my partner Yuki because when we collaborated, I understood the purpose of the function and the way that the conditionals and loops operated in it. I also liked sharing our projects with the rest of the class because we discussed the importance of including the range of all pixels, since otherwise it can distort the photo. This helped me realize that it's necessary to be precise with my inputs and review my work more carefully.
